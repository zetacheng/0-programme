# 0-programme

`0-programme` is the master coordination repository for a five-paper theoretical physics research programme.

This repository contains no complete paper manuscript. Detailed derivations, scientific code, results, and paper text remain in the relevant paper repository. This repository records only programme-level coordination and accepted cross-paper status.

## Paper repositories

1. [Paper 1 — Dark Matter Structure](https://github.com/zetacheng/1-dark-matter-structure)
2. [Paper 2 — Emergent Gravity](https://github.com/zetacheng/2-emergent-gravity)
3. [Paper 3 — Vector Sector](https://github.com/zetacheng/3-vector-sector)
4. [Paper 4 — Dark Energy and Cosmology](https://github.com/zetacheng/4-dark-energy-cosmology)
5. [Paper 5 — Topological Sector](https://github.com/zetacheng/5-topological-sector)

## Role separation

### ChatGPT

- conceptual discussion;
- ontology and physical interpretation;
- analytic derivation planning;
- gate and prompt design;
- cross-paper consistency analysis.

### Codex

- code, tests and reproducibility inside the relevant paper repo;
- programme-repo maintenance;
- synchronization reports;
- no promotion of results without review.

### Claude

- independent review;
- gate verdict;
- overclaim detection;
- paper updates after acceptance.

### Principal Investigator

- approves assumptions and scope;
- accepts or rejects verdicts;
- determines programme direction.

## Programme controls

Start with [MASTER_PROGRESS.md](MASTER_PROGRESS.md), [PAPER_MAP.md](PAPER_MAP.md), [MASTER_ROADMAP.md](MASTER_ROADMAP.md), and [SYNC_STATUS.md](SYNC_STATUS.md). Scientific evidence remains authoritative only in its owning paper repository.
